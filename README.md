# ☁️ 24h Cloud Cover Visual Simulator

TypeScript と HTML5 Canvas (2D) を用いて構築された、リアルタイムな雲量および時間経過の視覚シミュレーターです。外部ライブラリを一切使用せず、ピュアな数学的ノイズ（FBM / Value Noise）のみで自然な雲の表現を行っています。

## ✨ 主な機能

- **動的な雲量調整**: スライダーを動かすことで、快晴（0%）から全天の雲（100%）までリアルタイムに変化します。
- **24時間のタイムラプス**: 時間の経過とともに、深夜（星空） ➔ 朝焼け ➔ 昼（青空） ➔ 夕焼け ➔ 深夜 へと空と雲の色が線形補間（Lerp）によって美しく変化します。
- **立体感のある雲（逆光エッジ効果）**: 朝方や夕方には、太陽の光が雲のフチを透過・反射する「シルバーライニング効果」を疑似再現。
- **天候連動補正**: 雲量が90%を超えると、時間帯に関わらず全体が自動的にドス黒い曇天・雨雲のトーンへシフトします。
- **星のまたたき**: 夜間（雲のない部分）には、ピクセル固有のハッシュ値を利用して、個別に異なる周期でチカチカと光る星空が出現します。

## 🚀 動かし方

1. このリポジトリをクローンまたはダウンロードします。
2. `index.html` をブラウザで開きます。
   - *注意: ピクセルデータを直接書き換える処理（ImageData）を含んでいるため、ブラウザのセキュリティ制限を回避するために `Live Server` (VSCode拡張機能) や `npx serve` などのローカルサーバー経由での起動を推奨します。*
